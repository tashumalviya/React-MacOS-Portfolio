# React-MacOS-Portfolio
A modern MacOS-inspired portfolio built with React.js featuring an interactive desktop UI, dark mode, responsive design, and smooth user experience.

## Live Demo

🔗 https://your-project.vercel.app

## Features

- MacOS-inspired desktop UI
- Interactive Dock
- Application-style windows
- Resume Viewer
- Notes App
- CLI (Terminal-inspired)
- GitHub Profile Window
- Spotify Shortcut
- Live Date & Time Display
- Responsive Layout

## Tech Stack

- React.js
- Vite
- JavaScript (ES6+)
- HTML5
- SCSS
- React Icons
